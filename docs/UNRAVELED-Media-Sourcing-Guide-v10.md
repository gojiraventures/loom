## 1. The Sourcing Philosophy

**Their camera work. Our story.**

We need raw footage — archaeological sites, ancient landscapes, artifact close-ups, museum interiors, temple carvings, geological formations — WITHOUT someone else's narration or editorial framing baked in. The footage is the canvas. Our research layer is the painting.

**Priority order:**
1. Public domain / CC0 (free, no restrictions, no attribution required)
2. Creative Commons (free, attribution required, some share-alike)
3. Institutional open access (free, attribution required, non-commercial usually OK)
4. Unlimited subscription libraries (flat monthly fee, unlimited downloads)
5. Per-clip licensed footage (expensive, for hero shots only)
6. AI-generated atmospheric footage (for abstract/mood shots, never for "evidence")

**Hard rules:**
- Never present AI-generated footage as documentary evidence
- Always track licensing terms in our database per asset
- Archive original download receipts/licenses
- If footage has narration we disagree with, strip the audio and use video only — but only if the license permits derivative works

## 6. Original Footage — Phase-Gated Approach

**Phase 1 (pre-revenue): Zero-cost only.** Don't spend money proving the concept — prove it with free sources, then invest.

| Content | How to Get It | Cost | Phase |
|---------|--------------|------|-------|
| **Screen recordings of digital archives** | Record yourself navigating the Dead Sea Scrolls Digital Library, British Museum collection, CDLI, Sefaria, etc. This IS our footage — showing real primary sources on screen. | Free | **Phase 1** |
| **Our own interactive elements** | Screen-record the map, timelines, scripture comparisons, convergence cards from our own site. These are unique to us — nobody else has them. | Free | **Phase 1** |
| **YouTube CC-licensed footage** | Filter YouTube by Creative Commons license. Many academic channels and small documentary makers CC-license their work. Strip audio, use video only. | Free | **Phase 1** |
| **Google Earth / Google Arts & Culture** | Virtual flyovers of archaeological sites. 360° museum tours. Check Google's usage terms for each. | Free | **Phase 1** |
| **Interviews with researchers** | Remote via Riverside.fm or even Zoom. Audio-only is fine for podcast. Video is a bonus. | Free–$15/mo | **Phase 1** |
| **Museum artifact close-ups** | Many museums allow personal photography. Visit with a phone — modern phones shoot 4K. | Travel time only | **Phase 2** (after traction) |
| **Commissioned local videographers** | Hire via Fiverr/Upwork near Göbekli Tepe, Giza, Teotihuacan for B-roll. | $100-500/location | **Phase 3** (after revenue) |
| **Drone footage of ancient sites** | Commission drone operators on location. | $200-1000/location | **Phase 3** (after revenue) |

**The principle:** Don't commission footage until the content is proven to drive audience and revenue. The free sources above — especially screen recordings of real primary source databases — are more compelling than generic stock footage anyway. Showing someone the actual Gilgamesh Flood Tablet on the British Museum's website is more powerful than showing a stock shot of a generic cuneiform tablet.

## 8. Asset Management System

Every piece of media we download or license needs to be tracked in our Supabase database.

### media_assets table

```sql
CREATE TABLE media_assets (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  filename TEXT NOT NULL,
  media_type TEXT NOT NULL, -- 'video', 'image', 'audio', 'music', 'sfx'
  source_platform TEXT, -- 'storyblocks', 'artgrid', 'wikimedia', 'british_museum', etc.
  source_url TEXT, -- where we downloaded it from
  license_type TEXT NOT NULL, -- 'cc0', 'cc_by', 'cc_by_sa', 'cc_by_nc_sa', 'royalty_free', 'per_clip', 'ai_generated', 'original'
  attribution_required BOOLEAN DEFAULT FALSE,
  attribution_text TEXT, -- the required credit line
  commercial_use_allowed BOOLEAN DEFAULT TRUE,
  derivative_works_allowed BOOLEAN DEFAULT TRUE,
  expiry_date DATE, -- if license has an expiry
  storage_path TEXT, -- Supabase Storage path
  file_size_bytes BIGINT,
  resolution TEXT, -- '4K', 'HD', '8K'
  duration_seconds INTEGER, -- for video/audio
  has_audio BOOLEAN DEFAULT TRUE, -- for video: does it include audio we might want to strip?
  audio_type TEXT, -- 'narration', 'music', 'ambient', 'none'
  description TEXT,
  tags TEXT[], -- 'archaeological', 'landscape', 'artifact', 'aerial', 'underwater', 'atmospheric'
  topics TEXT[], -- which unraveled topics this asset relates to
  traditions TEXT[],
  used_in JSONB DEFAULT '[]', -- [{content_id, content_type, timestamp}]
  ai_generated BOOLEAN DEFAULT FALSE,
  download_date DATE DEFAULT CURRENT_DATE,
  downloaded_by TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_media_type ON media_assets(media_type);
CREATE INDEX idx_media_license ON media_assets(license_type);
CREATE INDEX idx_media_topics ON media_assets USING GIN(topics);
CREATE INDEX idx_media_tags ON media_assets USING GIN(tags);
```

### Why This Matters

If we use a Storyblocks clip in a YouTube video and later cancel our subscription, the perpetual license protects us. But we need the RECEIPT — the record of when we downloaded it under what license terms. If a rights holder ever challenges, our database IS our defense.

## 9. The "Footage Is Accurate But Narrative Is Not" Approach

This is the key insight. Much existing documentary footage was shot at real locations with real artifacts — the camera work is valid even when the voiceover is garbage. Our approach:

**Step 1:** Identify footage with good visuals but bad/irrelevant narration
**Step 2:** Check if the license allows derivative works (must allow remixing)
**Step 3:** Strip the audio entirely — use video track only
**Step 4:** Overlay our own narration, music, and data graphics
**Step 5:** Credit the original source as required by the license

**Example:** A YouTube creator films 20 minutes of gorgeous 4K footage inside the British Museum's Assyrian galleries with casual narration that gets several facts wrong. If they've CC-licensed it, we can:
- Use the video track (the Lamassu sculptures, the Gilgamesh tablet displays)
- Strip their audio completely
- Add our own ElevenLabs narration citing Andrew George's scholarship
- Add our data overlay showing the convergence with Ezekiel's Cherubim
- Credit the original videographer per the CC license

Their camera work. Our research layer. Everybody wins.
