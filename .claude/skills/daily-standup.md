# Daily Standup

Quick standup for Unraveled.ai:

1. **Done** — Run `git log --oneline -10`, summarize recent commits
2. **In progress** — Run `git status`, describe uncommitted work
3. **Next** — Read `/docs/UNRAVELED-Implementation-Handoff-v10.md` and identify the next uncompleted step
4. **Issues** — Any uncommitted changes at risk? Build errors? Broken links?

Keep it to 10 lines max. Morning check-in, not a deep review.
