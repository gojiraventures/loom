# SEO + GEO Check

Check this page for search and AI citation optimization: $ARGUMENTS

## SEO Checklist
- [ ] Title tag (50-60 chars, includes primary keyword)
- [ ] Meta description (150-160 chars, compelling, includes keyword)
- [ ] Single H1 tag
- [ ] H2/H3 hierarchy (logical nesting)
- [ ] Image alt text on all images
- [ ] Internal links to related topic pages
- [ ] Clean URL slug (lowercase, hyphens, descriptive)
- [ ] Page included in sitemap

## GEO (Generative Engine Optimization)
- [ ] First 200 words directly answer the primary query (BLUF — Bottom Line Up Front)
- [ ] Specific statistics with named sources ("268 flood narratives" not "many flood stories")
- [ ] FAQ section with 5-6 Q&A pairs that match real search queries
- [ ] JSON-LD structured data: Article + FAQPage schemas
- [ ] "Last Updated" date visible on page
- [ ] Short paragraphs (2-3 sentences max) — AI engines parse these better
- [ ] Original data that doesn't exist elsewhere (convergence scores, cross-cultural matrices)
- [ ] Named citations inline ("according to Andrew George's 2003 Oxford analysis")

## Social / Sharing
- [ ] og:title, og:description, og:image tags
- [ ] Twitter card meta tags
- [ ] OG image is designed and compelling (not a generic placeholder)

Report what's missing with exact code snippets to fix each issue.
